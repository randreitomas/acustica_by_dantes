# Acustica

Single-page MVP for **Acustica** — an intimate open-mic venue and coffeehouse.

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- shadcn/ui (Button, Card)
- Framer Motion

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Page flow

Navbar → Hero → Menu → About → Gallery → Reservations → Contact → Reviews → Online Ordering → Footer
