# Acustica

Single-page MVP for **Acustica by Dantes** — an intimate open-mic venue and coffeehouse in Sampaloc, Manila.

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion
- Lightweight UI primitives (Button, Card)

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Page flow

Navbar → Hero → Menu → About → Gallery → Reviews → Contact → Online Ordering → Footer
